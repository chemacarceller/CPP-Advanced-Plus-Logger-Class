# C++-Advanced-Plus-Logger-Class

MyLogger is an advanced plus C++ class for creating a log file.

It is designed to be used from a Python script.

This version is an enhanced version of the advanced version from the previous repository.

The "enhanced" part means that the class developed as a singleton in the advanced version is modified so that the singleton design pattern is applicable to threads where this class is instantiated.

This version would be suitable for scripts that are called recursively.

The QuickFolderSynchro application, written in Python for fast folder synchronization, uses this class in one of its versions.
